What is the CMF?

Set of loosely coupled bundles and uncoupled components concerened with building content mangement systems

PHPCR

  • Content Repository
  • Primary storage system used by the CMF
  • Stores content/data in a hierarchy like a filesystem
  • Allows SQL queries like an RDBMSM

CMF Routing

  • Dynamic Routing
  • Direct references to content

CMF Routing Auto Bundle

  • Automatically creates routes based on a URL schema
  • Handles conflicts
  • Handles defunct routes
  • URL schema: /posts/{date}/{title}
  • Save document "My Blog Post"
  • Creates route: /posts/2014-08-08/my-blog-post
